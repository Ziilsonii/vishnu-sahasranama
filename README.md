# 🪷 श्री विष्णु सहस्रनाम

**Interactive Web Application** featuring all **108 Shlokas** of the divine Vishnu Sahasranama from the Mahabharata (Anushasan Parva - Bhishma-Yudhishthira dialogue).

## Features
- ✨ Beautiful, serene dark theme with golden accents and Devanagari typography (Noto Serif Devanagari)
- 🔍 Powerful real-time search by shloka number, name, or meaning
- 👉 Click any card to expand and view detailed Hindi meanings for each name
- 📱 Fully responsive design optimized for mobile, tablet & desktop
- 🌸 Self-contained single HTML file - no dependencies

## 🚀 Live Site
**https://ziilsonii.github.io/vishnu-sahasranama/** (after enabling GitHub Pages)

## How to Enable GitHub Pages (Quick Setup)
1. Open your repository: https://github.com/Ziilsonii/vishnu-sahasranama
2. Go to **Settings** (tab at the top)
3. In the left sidebar, click **Pages**
4. Under "Build and deployment", set **Source** to **Deploy from a branch**
5. Select Branch: `main` and Directory: `/ (root)`
6. Click **Save**

The site will be live within a minute at the URL above!
